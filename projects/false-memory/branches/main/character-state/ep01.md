# 角色状态快照 — 第 1 集后

## 沈念
- 情绪状态：表面克制，内心动摇。专业自信出现第一道裂痕，夜晚独处时开始焦虑
- 知识范围：知道陆行声称记忆被篡改；看到了旧照片（自己、陆行、第三人）；被告知手腕疤痕来自车祸而非摔伤；发现相册有一整页照片被撕掉且自己毫无记忆
- 关系变化：对陆行从"疑似妄想症来访者"转为"无法轻易否定的人"；尚未与方远、小夏产生互动
- 关键行为：用 BPRS 筛查试探陆行（被识破）；独自翻查旧相册验证

## 陆行
- 情绪状态：偏执但克制，带有长期失眠的疲惫感。目标明确，情绪稳定
- 知识范围：持有旧照片（从"不该找到的地方"获得）；知道沈念手腕疤痕的真实来历；声称自己也在那场车祸中；知道有人篡改了记忆，但尚未明确指向谁
- 关系变化：主动接触沈念，从陌生人变为"留下名片等待回应"的状态
- 关键行为：闯入咨询室；出示照片；说出疤痕来历；留下名片后主动离开
