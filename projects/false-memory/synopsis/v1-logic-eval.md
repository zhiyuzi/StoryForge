### 逻辑一致性（4/5）

#### 硬事实清单

**时间线：**
- T-3年：方远主持"创伤后记忆干预"研究课题（伦理审批未通过但实际执行）
- T-3年：课题组野外调研，成员为沈念、陆行、方远、周然
- T-3年·事发日：沈念开车，山路暴雨，面包车失控翻覆
- T-3年·事发日：周然被甩出车外重伤，方远拨打120，周然在救护车到达前死亡
- T-3年·事发后：方远在医院对沈念和陆行实施催眠记忆干预，植入"什么都没发生"的替代记忆
- T-3年·事后：周然家属报警，案件被迅速以"自行离开"结案（方远疑似动用关系）
- T-3年·事后：沈念手机定位记录被手动删除，日程记录出现连续两周空白
- T=0（现在）：陆行25岁，自由职业者
- T=0：沈念为持证心理咨询师，方远为其督导/前导师

**角色知识范围：**
- 沈念（剧初）：不记得车祸，不认识陆行，以为手腕疤痕是小时候摔伤
- 陆行（剧初）：有噩梦和记忆碎片，知道车祸发生过但被所有人否认，持有旧照片
- 方远（全程）：知道全部真相——沈念开车、周然死亡、自己实施了记忆篡改
- 小夏（全程）：不知道真相，被方远利用为信息渠道，以为在保护沈念

**因果链：**
- 陆行的记忆封锁不完全（噩梦/碎片）→ 主动寻求沈念帮助
- 陆行持有旧照片 → 沈念无法否认关联 → 决定接诊
- 催眠回溯 → 逐步还原记忆碎片 → 发现催眠诱导指令 → 怀疑有人蓄意掩盖
- 调查方远课题 → 发现伦理审批未通过 → 怀疑方远
- 沈念自我催眠 → 看到"方远开车"的假记忆 → 第7集付费墙
- 第8集记忆矛盾（白衬衫 vs 深色外套）→ 信任崩塌 → 寻找客观证据
- 加油站监控 → 驾驶位是沈念 → 第9集真相反转

#### 逻辑断裂点

**瑕疵1：陆行的旧照片来源未交代**
- 位置：第1集
- 描述：陆行持有一张"他、沈念、和一个被划掉脸的第三人站在面包车前"的旧照片。如果方远对陆行实施了完整的记忆篡改并持续监控三年，为什么这张关键物证没有被销毁？方远删除了沈念的手机定位、压下了警方调查，却遗漏了一张直接关联所有当事人的照片，这在方远"谨慎监控者"的人设下略显不合理。
- 严重程度：小瑕疵。可以通过补充说明来解释（如：照片是陆行在记忆碎片驱动下从某个方远不知道的地方找到的，或者是陆行在被催眠前藏起来的）。

**瑕疵2：沈念自我催眠还原出"错误记忆"的机制需要更多铺垫**
- 位置：第7集
- 描述：沈念通过自我催眠看到"方远满身酒气坐在驾驶位"的画面，第8集揭示这是错误的（实际开车的是沈念自己）。问题在于：如果方远植入的替代记忆是"什么都没发生"，那沈念自我催眠突破封锁后，应该还原的是真实记忆（自己开车），而不是一个全新的错误记忆（方远开车）。这个"方远开车"的画面从何而来？
- 可能的解释：（a）方远植入了多层记忆——第一层是"什么都没发生"，第二层是"如果你突破了第一层，你会看到是我开的车"，作为安全网；（b）沈念的潜意识出于自我保护，在突破封锁时自动将"自己开车"替换为"方远开车"。梗概中没有明确交代是哪种机制。这是全剧最关键的反转支点，需要在正式剧本中给出合理解释。
- 严重程度：中等瑕疵，但不构成逻辑断裂。心理悬疑赛道允许记忆的不可靠性，且第8集的"你确定这就是真相吗？"字幕已经暗示了这一层。建议在剧本阶段通过方远的台词或沈念的专业分析补充解释。

**整体判断：**
时间线自洽，角色知识范围合理（方远全知、沈念和陆行逐步解锁、小夏始终蒙在鼓里），因果链完整且每个发现都有合理的触发原因。世界观设定（通过催眠/心理暗示篡改记忆）在现实框架内基本可信——方远作为资深心理学教授，对处于急性应激状态的学生实施催眠干预，在专业能力上说得通。两处瑕疵均为可修补的小问题，不影响整体逻辑架构。

**评分理由：** 整体通顺，存在1处需要补充说明的小瑕疵（照片来源）和1处需要在剧本阶段强化铺垫的中等问题（错误记忆机制），但无硬性逻辑断裂。给4分。

### 付费点优化（5/5）

#### 付费墙位置分析
- 付费墙设在第7集结尾，10集剧共30分钟，付费墙在第21分钟处，免费内容占比70%。

#### 付费墙前张力曲线
- 第1集：建立谜题（照片+疤痕），张力启动
- 第2集：催眠揭示"你也在车上"，张力跳升
- 第3集：发现催眠诱导指令+四人报名表，张力持续攀升
- 第4集：伦理审批未通过+陆行情绪崩溃+"有人死了"，张力再升
- 第5集：沈念自我催眠听到方远的声音+第四个人可能已死，张力高位
- 第6集：周然身份浮出+方远微表情破绽+U盘失踪，张力逼近顶点
- 第7集：小夏真相+师徒心理博弈+沈念突破记忆封锁看到"真相"，张力达到最高点

张力曲线呈持续上升态势，每集都有新的信息炸弹推高张力，没有回落或平台期。

#### 截断点评估
第7集结尾的截断极其精准：
1. 沈念刚刚"看到"了最震撼的画面（方远酒驾导致周然死亡）
2. 观众以为真相已经揭晓
3. 但黑屏字幕"你确定这就是真相吗？"直接制造认知失调——观众必须知道这到底是不是真相
4. 这是一个双重钩子：既有情感冲击（导师是杀人者），又有悬念反问（可能还有反转）

#### 付费后兑现评估
第8集开场立即兑现：
- 陆行指出记忆矛盾（白衬衫 vs 深色外套）→ 直接推翻第7集的"真相"
- 信任崩塌 → 新的冲突维度
- 加油站监控 → 驾驶位是沈念自己 → 超出观众预期的二次反转
- 信息量极大，远超观众付费前的预期

**评分理由：** 付费墙设在全剧张力最高点，截断方式同时提供情感冲击和悬念反问，形成双重付费驱动。付费后第一集立即兑现且超额兑现（不仅揭示真相，还推翻了观众以为的真相）。完全符合 LR-003（付费墙前最后一集的结尾钩子必须是全剧最强的）。给5分。

#### 人设标签检查

| 人设标签 | 匹配情况 | 说明 |
|---|---|---|
| 主角：侦探/记者/普通人被卷入 | ✓ | 沈念是心理咨询师，从职业责任出发被卷入，符合"专业人士被卷入"的变体 |

| 真凶：最不像凶手的人 | 部分符合 | 最终"真凶"是沈念自己（开车者），确实是"最不像凶手的人"（主角即肇事者）。但方远从第4集开始就被强烈暗示为嫌疑人，中间缺少有效的"洗白"环节来维持悬念——观众可能过早锁定方远 |

#### 可强化之处
- 嫌疑人池偏窄：建议在剧本阶段为陆行增加一些可疑行为（如他的记忆碎片中有自相矛盾之处），让观众在沈念、方远、陆行三人之间摇摆，增强猜测空间。
- 方远的嫌疑建立过于线性：从第4集开始，几乎每集都在加强方远的嫌疑，缺少"方远可能是无辜的"的反向线索来制造摇摆。建议在第5或第6集加入一个方远看似被洗白的情节，让观众的判断来回摆动。

**评分理由：** 核心套路完全匹配，必备元素全部到位，经典结构严格遵循，线索公平性好。主角人设标签清晰可识别。扣1分原因：嫌疑人池偏窄，方远嫌疑建立过于线性缺少摇摆，这在悬疑赛道中会略微削弱观众的猜测乐趣。给4分。
