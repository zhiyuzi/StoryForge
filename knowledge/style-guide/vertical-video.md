# 竖屏构图规范（Vertical Video Guide）

## 画幅比例
- 9:16 竖屏（1080×1920）
- 所有构图以竖屏为基准

## 安全区
```
┌──────────────┐
│   状态栏区    │ ← 顶部 5% 避免放重要内容
│              │
│   主要内容区  │ ← 中间 70% 是核心区域
│   （主体居中  │
│    偏上 1/3） │
│              │
│   字幕区     │ ← 底部 15% 留给字幕
│   操作区     │ ← 底部 10% 可能被 UI 遮挡
└──────────────┘
```

## 构图原则

### 人物构图
- 单人：主体居中，头部在上 1/3 线
- 双人对话：左右分布，或前后景深
- 群戏：利用纵深排列，避免横排（竖屏横向空间有限）

### 景别与竖屏适配
| 景别 | 竖屏适配 | 注意事项 |
|---|---|---|
| 远景 | 利用纵深感 | 横向场景需裁切或重构 |
| 全景 | 人物占画面 60% | 不要留太多头顶空间 |
| 中景 | 最常用，腰部以上 | 手势和道具要在画面内 |
| 近景 | 胸部以上 | 情绪表达主力 |
| 特写 | 面部或道具 | 竖屏特写冲击力强 |

### 运镜适配
- 推拉：竖屏效果好，纵深感强
- 横摇：慎用，竖屏横向空间有限，容易丢失主体
- 跟拍：适合走廊、楼梯等纵向空间
- 固定：对话场景首选

## 字幕规范
- 位置：底部 15% 区域
- 每行不超过 15 字
- 字体大小：确保在手机上清晰可读
- 背景：半透明黑底或描边，确保可读性

## 转场
- 硬切：最常用，节奏快
- 叠化：时间流逝、回忆
- 闪白/闪黑：冲击、反转
- 避免花哨转场（擦除、旋转等）
