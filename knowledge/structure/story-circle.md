# Dan Harmon Story Circle（故事圆环）

## 概述
Dan Harmon 的故事圆环是一个 8 步叙事结构，适用于任何故事。在短剧中，可以将整个圆环压缩到 1 集或展开到全剧。

## 8 个步骤

```
        1. 舒适区
       ╱         ╲
  8. 改变          2. 需求
     │                │
  7. 回归          3. 出发
     │                │
  6. 得到          4. 适应
       ╲         ╱
        5. 找到
```

1. **舒适区（You）**：角色在日常环境中
2. **需求（Need）**：角色想要或需要某样东西
3. **出发（Go）**：角色进入不熟悉的环境
4. **适应（Search）**：角色适应新环境，面对挑战
5. **找到（Find）**：角色得到想要的东西
6. **得到（Take）**：但付出了代价
7. **回归（Return）**：角色回到熟悉的环境
8. **改变（Change）**：角色已经不同了

## 短剧适配

### 单集应用（微圆环）
每集可以是一个微型圆环：
- 开头：角色在某个状态（1-2）
- 中段：面对挑战，做出选择（3-5）
- 结尾：状态改变，但新问题出现（6-8 + 下集钩子）

### 全剧应用（宏圆环）
10 集短剧的宏观结构：
- 第 1-2 集：舒适区 → 需求（建立世界，触发事件）
- 第 3-4 集：出发 → 适应（进入新状态，面对挑战）
- 第 5-6 集：找到（获得关键资源/信息）
- 第 7-8 集：得到 → 代价（最大危机，付费墙）
- 第 9-10 集：回归 → 改变（解决问题，角色成长）
