# Save the Cat 节拍表（短剧适配版）

## 概述
Blake Snyder 的 Save the Cat 节拍表是好莱坞最常用的故事结构。这里是适配短剧（10 集 × 2 分钟）的版本。

## 15 个节拍 → 短剧映射

| 原始节拍 | 短剧位置 | 说明 |
|---|---|---|
| Opening Image | 第 1 集开头 | 主角的"之前"状态 |
| Theme Stated | 第 1 集 | 暗示主题（通常通过对白） |
| Set-Up | 第 1-2 集 | 建立世界、角色、问题 |
| Catalyst | 第 2 集 | 触发事件（改变一切的瞬间） |
| Debate | 第 2-3 集 | 主角犹豫是否行动 |
| Break into Two | 第 3 集 | 主角做出决定，进入新世界 |
| B Story | 第 3-4 集 | 感情线/副线启动 |
| Fun and Games | 第 4-6 集 | 核心卖点展示（打脸/甜蜜/推理） |
| Midpoint | 第 5 集 | 中点反转（假胜利或假失败） |
| Bad Guys Close In | 第 6-7 集 | 对手反击，压力增大 |
| All Is Lost | 第 7-8 集 | 最低谷（付费墙前） |
| Dark Night of the Soul | 第 8 集 | 主角最绝望的时刻 |
| Break into Three | 第 8-9 集 | 主角找到解决方案 |
| Finale | 第 9-10 集 | 最终对决，解决问题 |
| Final Image | 第 10 集结尾 | 主角的"之后"状态 |

## 短剧特殊调整
- Fun and Games 阶段是短剧的核心卖点区，要密集展示赛道特色
- All Is Lost 必须在付费墙前，这是付费转化的关键
- 短剧没有时间做长篇 Debate，主角的犹豫要快速解决
- B Story（感情线）不能喧宾夺主，要服务于主线
