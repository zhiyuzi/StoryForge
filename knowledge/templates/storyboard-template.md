# 分镜输出模板（Storyboard Template）

## 使用说明
分镜生成 Agent 必须严格按照此模板输出每集分镜。

---

```markdown
# 第 {N} 集分镜：{本集标题}

## 集信息
- 时长：{duration} 秒
- 镜头数：{shot_count}
- 核心事件：{一句话概括}

---

## 镜头 1
| 项目 | 内容 |
|---|---|
| 镜号 | 001 |
| 景别 | {远景/全景/中景/近景/特写} |
| 运镜 | {固定/推/拉/摇/移/跟} |
| 画面描述 | {画面内容，以摄影机视角描述} |
| 构图要点 | {竖屏构图说明，主体位置} |
| 对白/旁白 | {角色名}：{台词} 或 （旁白）{内容} 或 无 |
| 音效/音乐 | {环境音/配乐提示} |
| 时长 | {x} 秒 |
| 情绪 | {本镜头的情绪基调} |

---

## 镜头 2
（同上结构）

---

## 镜头 {M}（结尾）

（同上结构）

**【本集视觉钩子】**
{描述最后一个镜头如何制造视觉悬念}

---

## 本集摘要
- 核心推进：{本集推进了什么}
- 视觉重点：{本集的视觉亮点}
- 角色变化：{哪些角色状态发生了变化}
- 新增伏笔：{本集埋下的伏笔}
- 解决伏笔：{本集解决的伏笔}

## 技术备注
- 总时长：{sum} 秒
- 景别分布：远景 {n}、全景 {n}、中景 {n}、近景 {n}、特写 {n}
- 特殊拍摄需求：{如有}
```
