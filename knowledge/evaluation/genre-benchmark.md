# 各赛道基准分（Genre Benchmark）

## 用途
为不同赛道的评估提供基准参考。不同赛道的侧重点不同，评估时应参考对应赛道的基准。

## 基准分表

### 复仇/逆袭赛道

| 维度 | 基准分 | 权重 | 说明 |
|---|---|---|---|
| 格式规范 | 通过 | - | 确定性检查 |
| 叙事效率 | 4/5 | 高 | 复仇剧节奏要快 |
| 冲突处理 | 4/5 | 极高 | 核心卖点 |
| 角色一致性 | 3/5 | 中 | 基本一致即可 |
| 情感深度 | 3/5 | 中 | 不是主要卖点 |
| 逻辑一致性 | 3/5 | 中 | 基本合理即可 |
| 爽点密度 | 4/5 | 极高 | 核心卖点 |
| 钩子强度 | 4/5 | 高 | 留存关键 |
| 付费点优化 | 4/5 | 高 | 商业关键 |
| 赛道匹配度 | 4/5 | 高 | 必须符合套路 |
